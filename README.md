# Lima Fit Method - Landing Page

Arquitectura estática construida para rendimiento extremo.
* Framework: Astro
* Estilos: Tailwind CSS v4
* Interactividad: Swiper JS (Vanilla)

## Despliegue Local

1. Clonar el repositorio:
   `git clone https://github.com/tu-usuario/landingpage-trainer.git`
2. Instalar dependencias:
   `npm install`
3. Levantar el servidor de desarrollo:
   `npm run dev`
